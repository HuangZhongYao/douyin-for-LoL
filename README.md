#### 运行教程

1.  clone 项目到HBuilder 中打开
2.  修改 App.vue 中 baseApi 后端接口地址
3.  点击HBuilder运行到浏览器选择Chrome
4.  使用手机访问项目或者浏览器调成移动设备模式
